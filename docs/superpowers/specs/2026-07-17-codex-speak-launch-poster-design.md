# Codex Speak Launch Poster Design

## Goal

Create two coordinated launch posters that explain Codex Speak's strongest
advantage at a glance: its speech pipeline stays on the Mac and uses the native
macOS `say` command instead of an external TTS service.

The posters must not imply that the entire Codex conversation is offline. The
privacy claim applies only to Codex Speak's speech processing, queueing, and
playback pipeline.

## Deliverables

1. GitHub poster: landscape 16:9, English-first, suitable for a README hero or
   release announcement.
2. Xiaohongshu poster: portrait 3:4, Chinese-first, suitable as a post cover.

Both final raster images will be saved under `assets/posters/` with descriptive
filenames. They form one visual family but use channel-specific layouts rather
than cropping a single composition.

## Visual Direction

- Premium, minimal macOS-product aesthetic.
- Deep charcoal-black background with restrained indigo and violet light.
- Use the production Codex Speak speaker prompt logo from
  `assets/codex-speak-github.png` as a reference and preserve its speaker
  silhouette, embedded `>_` prompt, colors, and proportions.
- One dominant logo, one headline, one proof line, and a compact three-mode
  control motif.
- Add subtle waveform or menu-bar cues only when they support the hierarchy.
- Avoid stock imagery, people, extra logos, decorative 3D clutter, fake app
  screenshots, watermarks, or claims not supported by the product.

## GitHub Poster

### Layout

- 16:9 landscape composition.
- Large logo balanced against left-aligned product copy.
- Generous negative space and strong readability at README width.

### Exact Copy

- `CODEX SPEAK`
- `Your Codex speaks. Your voice stays on your Mac.`
- `macOS say · No TTS API · No API Key · No Network`
- `Silent · Summary · Full`
- `Local refers to the Codex Speak speech pipeline.`

## Xiaohongshu Poster

### Layout

- 3:4 portrait composition.
- Logo and headline dominate the top two thirds.
- Proof points sit in a compact lower information panel.
- Typography remains readable as a mobile feed cover.

### Exact Copy

- `CODEX SPEAK`
- `让 Codex 开口说话`
- `语音完全留在你的 Mac`
- `只用 macOS say`
- `无需 TTS API · 无需 API Key · 不联网`
- `静音 · 摘要 · 全文`
- `本地仅指 Codex Speak 的语音处理与播放链路`

## Acceptance Criteria

- Both posters look like one brand family.
- The production logo remains recognizable and is not redesigned.
- All required text is rendered verbatim with no extra copy.
- Privacy language is accurate and does not claim Codex itself is fully local.
- The GitHub poster reads clearly in landscape; the Xiaohongshu poster reads
  clearly on a phone.
- No watermark or unrelated brand mark appears.
